# Fluttershy's Magical World Website

A beautiful, interactive website themed around Fluttershy from My Little Pony, designed for your daughter!

## Features

- 🌈 **White-based rainbow shader background** - Animated gradient that shifts through soft pastel colors
- 🦋 **Floating butterflies** - Interactive butterfly animations
- ✨ **Sparkle effects** - Magical sparkle animations
- 📱 **Responsive design** - Works on all devices
- 🎨 **Fluttershy color scheme** - Pink, peach, and pastel colors

## How to Use

1. **Open the website**: Simply open `index.html` in any web browser
2. **Add Fluttershy image**: Replace the placeholder pony image with your own Fluttershy picture:
   - Find a Fluttershy image you like
   - Save it as `fluttershy.png` in the same folder
   - Edit `styles.css` and update the `.pony-image` class to use your image

## Customization

### Change the rainbow colors
Edit the `styles.css` file and modify the gradient colors in the `.rainbow-shader` class.

### Adjust animation speed
Change the `15s` value in `animation: rainbowShift 15s ease infinite;` to make the rainbow move faster or slower.

### Add more features
You can extend this website by:
- Adding more Fluttershy quotes
- Including games or activities
- Adding a photo gallery
- Creating interactive elements
